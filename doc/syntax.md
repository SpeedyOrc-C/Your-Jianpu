# 文法

```
1 1 5 5 | 6 6 5 - |
4 4 3 3 | 2 2 1 - ||
```

```
7// 6// #5// 6// | 1' 2'// 1'// 7// 1'// 3' 4'// 3'// #2'// 3// |
7// 6// #5// 6// 7// 6// #5// 6// 1 6/ 1/ |
```

几分音符
```
1 1/ 1// 1///
```

高八度
```
1 1' 1'' 1'''
```

低八度
```
1 1. 1.. 1...
```

附点
```
1 1* 1** 1***
```

升降号
```
1# 1b
```


三连音
```
<3> 1 2 3 </3>
```

圆滑线与连音线
```
(1 2 3)
```

装饰音
```
[7 6 5# 6] 1'
```

小节线
```
| |: :| ||
```
